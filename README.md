
# npm-bootstrap

npm based frontend boilerplate for Twitter Bootstrap and jQuery

### Demo


	example for Twitter Bootstrap scrollspy, affix, etc.

Installation
------------

	git checkout demo-1
    npm install 

Usage
-----

	npm start

Then open browser at http://localhost:3000

### Dev Dependencies

* browserify
* copyfiles
* html-minifier
* live-server
* nodemon
* stylus
* uglify-js

### Dependencies

* bootstrap
* browsernizr
* jquery
